- 语义化标签：`header`、`footer`、`article`等
- Audio、Video API
- `localStorage`、`sessionStorage`本地缓存
- Canvas
- 地理位置API Geolocation
- WebWorker
- Input 表单控件的类型

  6.  range
  7.  search
  8.  month
  9.  week
  10. time
  11. url
